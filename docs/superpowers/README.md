# Agent 执行规范目录

## 用途

本目录服务于 AI Agent 执行任务，不是普通项目说明文档。这里记录规格、执行计划、任务卡和复盘结果，帮助不同 Agent 在明确边界内协作。

## 应放内容

- `specs/`：任务或阶段的规格说明。
- `plans/`：可执行计划，说明步骤、文件范围和验证方式。
- `tasks/`：具体任务卡，适合交给执行 Agent。
- `reviews/`：任务完成后的检查记录。

## 当前状态

当前已初始化 Phase 0 文档体系相关规范。

## 后续维护规则

- 所有执行任务必须先有 plan 或 task。
- 每个任务必须声明允许修改和禁止修改的文件。
- 本目录内容应面向 Agent 执行，避免写成泛泛的人类阅读介绍。
