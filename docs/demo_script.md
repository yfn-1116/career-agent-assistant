# Demo 演示脚本（组员 B 用）

## 打开方式

浏览器输入：
```
http://localhost:8501
```
（或外网地址 `https://warm-declare-medal-altered.trycloudflare.com`）

---

## 第 1 步：展示 GitHub 仓库查询（30 秒）

### 输入
```
你能查看我的GitHub仓库吗 https://github.com/yfn-1116
```

### 预期结果
列出所有公开仓库：career-agent-assistant、chem-auto-titration、polyu-internship-project、opencode 等。

### 讲解词
"先看第一个功能——GitHub 仓库查询。粘贴一个 GitHub 用户链接，系统自动列出该用户的所有公开仓库，包括项目名、语言、描述。不需要登录，不需要 Token。"

---

## 第 2 步：查看具体仓库内容（30 秒）

### 输入
```
https://github.com/yfn-1116/career-agent-assistant这个仓库做什么的
```

### 预期结果
返回该仓库的 README 内容：项目定位、核心能力、技术栈、快速运行说明。

### 讲解词
"粘贴具体仓库链接，系统实时抓取 README 内容。这是直接从 GitHub 线上读的，不是本地缓存。可以看到项目定位、技术栈、运行方式——全部真实数据。"

---

## 第 3 步：查看知识库中的项目经历（20 秒）

### 输入
```
我有哪些项目经历？
```

### 预期结果
列出知识库中索引的项目：智能投递辅助 Agent、自动滴定系统、PolyU Smart Journey 等。

### 讲解词
"系统维护了一个个人知识库——简历、项目经历、技能描述都索引在里面。问一句话就能检索出所有相关经历。"

---

## 第 4 步：JD 匹配分析（核心功能，1.5 分钟）

### 输入（粘贴以下完整 JD）
```
Agent开发实习生 (140-190元/天)
工作地点：深圳
实习要求：5天/周，最少3个月
最低学历：本科在读
技术标签：Java、C++、Python
职位描述
希望有 AI 相关的开发经验
岗位职责：
负责 AI Agent 系统的开发和优化，包括记忆模块、规划模块和工具调用模块；
开发和完善基于大语言模型（LLM）的智能体系统，提升 Agent 的自主决策和任务执行能力；
设计并实现 Agent 的工具使用接口，实现与各类外部系统和 API 的集成。
优化 Agent 的对话系统，提升用户交互体验和任务完成效率。
开发 Agent 的知识库管理系统，实现高效的信息存储和检索。
任职要求：
本科及以上学位在读，计算机科学、人工智能或相关专业优先；
熟悉 Python、Java 等主流编程语言，具备良好的代码风格和工程实践；
熟悉大语言模型（LLM）的原理和应用，有 GPT 等模型应用经验；熟悉机器学习、深度学习等 AI 技术，了解最新研究进展；
具备优秀的问题解决能力和系统架构设计能力；
具备良好的团队协作能力和沟通能力。
```

### 等待
15-20 秒（Qwen 大模型在做匹配分析）。

### 预期结果
返回完整的匹配分析报告：
- **匹配度**：85%
- **强项**：具体说明哪些项目覆盖了 JD 的哪些要求
- **弱项**：Java 缺失、机器学习深度不足
- **简历要点**：3 条 STAR 法则描述，每条标注证据来源
- **HR 沟通话术**：一段可复制的打招呼文本

### 讲解词
"这是最核心的功能——JD 匹配分析。粘贴一段完整的岗位描述，系统调用大模型做深度分析。它不是简单的关键词匹配，而是理解 JD 的语义、对比知识库中的真实项目经历、用 STAR 法则生成可以直接写进简历的描述。注意看——每条简历要点都标注了证据来源，不会编造经历。"

---
