# 知识库笔记目录

## 用途

本目录用于沉淀 RAG、Agent 工程、LangGraph、Prompt Pattern 和开源项目调研等学习笔记。

## 应放内容

这里应放概念解释、实践经验、对项目有帮助的模式和反模式，以及成熟开源项目的参考分析。

## 当前状态

当前已包含 RAG、Agent、LangGraph、Prompt Pattern 和开源参考项目调研入口。

## 重点文档

- `05-open-source-reference-projects.md`：记录 LangGraph、Dify、RAGFlow、DeerFlow、OpenHands、Khoj、Flowise 等项目的参考设计和裁剪结论。

## 后续维护规则

- 笔记可以是探索性的，但必须标注是否已经在项目中采用。
- 不把笔记内容直接等同于项目决策。
- 开源项目调研只能总结可借鉴设计，不复制大段外部内容。
