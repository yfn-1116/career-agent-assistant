### 1. career-agent-assistant
- URL: https://github.com/chenxiaoming-dev/career-agent-assistant（示例）
- 语言: Python
- Stars: 15
- 描述: 基于 RAG 的实习投递辅助 Agent。将个人经历文档化（Markdown），使用 Chroma 向量检索 + LLM 做 JD 匹配分析，Streamlit 做轻量 Demo 展示。
- 技术栈: Python, LangChain, Chroma, Streamlit, DeepSeek API
- 个人贡献:

- 适配岗位: AI 应用开发、RAG 工程师、大模型应用实习生
