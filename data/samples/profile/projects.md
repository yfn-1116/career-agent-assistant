## 项目 1：智能投递辅助 Agent

- **类型**：AI 应用 / RAG 系统
- **背景**：找实习时发现手动匹配 JD 和自身经历效率很低——对着 10 份 JD，需要反复翻看自己的项目文档来回忆哪些经历能对上。于是希望做一个工具，把自己的经历文档化，然后用 AI 做匹配分析。
- **技术栈**：Python + LangChain + Chroma + Streamlit + DeepSeek API
- **个人负责**：独立开发，负责全部设计与实现
- **核心亮点**：
  - 将自己的项目、技能、实习经历写成结构化 Markdown 文档，作为 RAG 知识库
  - 使用 Chroma 向量数据库存储文档 embedding，基于 LangChain 搭建检索 pipeline
  - 设计 Prompt 模板，让 LLM 基于检索到的经历做 JD 匹配分析，输出匹配度、优势项和不足项
  - 使用 Streamlit 搭建轻量前端，支持上传 JD、查看检索证据和匹配分析结果
- **可匹配岗位**：AI 应用开发、RAG 工程师、大模型应用实习生
- **可作为简历表达的依据**：
  - "独立开发基于 RAG 的实习投递辅助工具，将个人经历文档化并实现 JD 智能匹配分析"
  - "使用 LangChain + Chroma 搭建检索增强生成 pipeline，实现基于个人知识库的匹配问答"
  - "设计结构化 Prompt 模板，控制 LLM 基于真实经历输出匹配结果，避免编造"

## 项目 2：自动滴定系统

- **类型**：实验室自动化 / 计算机视觉
- **背景**：在化学系实验室做本科生研究助理时，发现手动滴定操作重复且容易因视觉疲劳导致终点判断不一致。希望用摄像头 + 图像处理实现自动终点检测。
- **技术栈**：Python + OpenCV + Arduino + 3D 打印
- **个人负责**：
  - 设计图像处理流程（颜色空间转换、ROI 提取、颜色变化检测）
  - 编写 Python 控制脚本，通过串口与 Arduino 通信控制滴定阀
  - 3D 打印滴定管支架和摄像头固定装置
- **核心亮点**：
  - 使用 OpenCV 实时分析溶液颜色变化，自动判定滴定终点
  - RGB → HSV 颜色空间转换提高了不同光照条件下的检测稳定性
  - Arduino 控制步进电机驱动滴定阀，实现精确滴加速度控制
  - 在 30 次标准样品滴定测试中，终点判定一致性优于手动操作（RSD < 0.5%）
- **可匹配岗位**：Python 开发、自动化测试、计算机视觉实习生
- **可作为简历表达的依据**：
  - "开发基于 OpenCV 的自动滴定终点检测系统，将图像处理技术应用于化学实验自动化"
  - "使用 Python + Arduino 实现传感器数据采集与设备控制，完成硬件-软件闭环系统"
  - "设计 HSV 颜色空间转换方案，提高了不同光照条件下颜色检测的鲁棒性"

## 项目 3：PolyU Smart Journey（实习自做系统）

- **类型**：Web 应用 / 全栈开发
- **背景**：在 PolyU 暑期实习期间，参与开发一个校园路径规划 Web 应用。用户输入起点和终点，系统返回最优路径并标注沿途设施（食堂、打印店、自习室等）。
- **技术栈**：FastAPI + Leaflet.js + PostgreSQL + Docker
- **个人负责**：
  - 开发 3 个后端 API 接口（路径查询、设施查询、用户反馈）
  - 设计数据库表结构（建筑、路径、设施、用户反馈）
  - 编写 Docker Compose 配置文件，实现一键部署
- **核心亮点**：
  - 使用 A* 算法实现校园路径规划，结合设施数据提供 "顺路" 推荐
  - FastAPI 接口响应时间 < 200ms（P95）
  - 使用 Leaflet.js 实现交互式地图展示
  - Docker 部署使得环境配置从 30 分钟减少到 1 条命令
- **可匹配岗位**：后端开发、全栈开发、Python 实习生
- **可作为简历表达的依据**：
  - "使用 FastAPI 开发 RESTful API 接口，P95 响应时间 < 200ms"
  - "设计 PostgreSQL 数据库表结构，支持路径查询和设施推荐功能"
  - "使用 Docker 容器化部署，将环境搭建时间从 30 分钟缩短至 1 条命令"

## 项目 4：Dog Breed Identification CNN 实验

- **类型**：机器学习 / 图像分类
- **背景**：机器学习课程大作业，使用 CNN 对 120 种犬类进行品种识别。数据集来自 Stanford Dogs Dataset。
- **技术栈**：Python + PyTorch + CNN（ResNet-18）+ Jupyter
- **个人负责**：
  - 数据预处理（图像增强、归一化、训练/验证/测试集划分）
  - 使用 ResNet-18 预训练模型做迁移学习
  - 超参数调优和实验结果分析
  - 撰写实验报告
- **核心亮点**：
  - 使用迁移学习（ResNet-18 pre-trained on ImageNet），在 120 类上达到 78% Top-1 准确率
  - 实现了数据增强 pipeline（随机裁剪、水平翻转、颜色抖动）提升泛化能力
  - 对比了不同优化器（SGD、Adam）和学习率调度策略的效果
  - 使用混淆矩阵和 Grad-CAM 可视化分析模型行为
- **可匹配岗位**：机器学习实习生、数据分析实习生
- **可作为简历表达的依据**：
  - "基于 PyTorch 实现 CNN 图像分类模型，使用迁移学习在 120 类犬种识别任务上达到 78% 准确率"
  - "设计数据增强 pipeline 和超参数调优策略，提升了模型泛化能力"
  - "使用 Grad-CAM 可视化分析模型决策依据，理解 CNN 特征提取行为"
