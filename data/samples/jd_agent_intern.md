## JD 1：AI Agent 开发实习生

### 基本信息
- 公司：XX 智能科技（示例）
- 岗位：AI Agent 开发实习生
- 地点：北京
- 类型：实习（每周 4 天以上，至少 3 个月）
- 薪资：200-300 元/天

### 岗位职责
1. 参与公司 AI Agent 平台的核心模块开发，包括 Agent 编排、工具集成、状态管理
2. 基于 LangChain / LangGraph 构建和优化多 Agent 协作流程
3. 参与 RAG 系统的开发和优化，提升检索质量和生成效果
4. 编写技术文档和单元测试，保证代码质量

### 任职要求
1. 计算机相关专业本科及以上学历，2025 年或 2026 年毕业
2. 熟练掌握 Python，有实际项目开发经验
3. 了解 LangChain、LangGraph 等 AI 应用开发框架
4. 了解 RAG（检索增强生成）的基本原理和实现方式
5. 了解向量数据库（如 Chroma、FAISS、Pinecone 等）的基本使用
6. 有 FastAPI、Streamlit 等 Web 框架使用经验者优先
7. 有 GitHub 开源项目或技术博客者优先
8. 具备良好的学习能力和团队协作能力

### 加分项
- 有多 Agent 系统或 Agent 编排相关项目经验
- 了解 Docker 容器化部署
- 参加过 AI 相关竞赛或 Hackathon
- 了解 Prompt Engineering 和 LLM 评估方法
