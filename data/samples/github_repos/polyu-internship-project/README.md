# polyu-internship-project

## 项目背景

在 PolyU 暑期实习期间，参与校园路径规划 Web 应用开发。用户输入起点和终点，返回最优路径并标注沿途设施。

## 技术栈

FastAPI, Leaflet.js, PostgreSQL, Docker

## 核心模块

- 后端 API：路径查询、设施查询、用户反馈 3 个接口
- 路径规划：A* 算法 + 设施顺路推荐
- 数据库：PostgreSQL 存储建筑、路径、设施数据
- 部署：Docker Compose 一键部署

## 当前完成状态

开发完成，API 响应 P95 < 200ms。

## 可匹配岗位方向

后端开发、全栈开发、Python 实习生

## 简历表达依据

使用 FastAPI 开发 RESTful API，Docker 部署，数据库设计，完整项目开发经验。
