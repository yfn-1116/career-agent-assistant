# chem-auto-titration

## 项目背景

化学实验室手动滴定操作重复且容易因视觉疲劳导致终点判断不一致。使用摄像头 + 图像处理实现自动终点检测。

## 技术栈

Python, OpenCV, Arduino, 3D Printing

## 核心模块

- 图像采集：USB 摄像头实时捕获溶液颜色
- 颜色分析：RGB → HSV 空间转换，ROI 提取，颜色变化检测
- 设备控制：Arduino 步进电机驱动滴定阀
- 终点判定：颜色变化阈值触发自动停止

## 当前完成状态

原型完成，30 次标准样品测试 RSD < 0.5%。

## 可匹配岗位方向

Python 开发、计算机视觉、实验室自动化

## 简历表达依据

开发基于 OpenCV 的自动滴定终点检测系统，将图像处理技术应用于化学实验自动化。
