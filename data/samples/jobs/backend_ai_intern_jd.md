# 岗位 JD：后端开发实习生（AI 方向）

> 说明：本文档为脱敏示例 JD。

## 基本信息

- 公司：WW 网络科技（示例，约 300 人，SaaS 平台方向）
- 岗位：后端开发实习生
- 地点：深圳
- 类型：实习（每周 4 天以上，至少 4 个月）

## 岗位职责

1. 参与 AI 产品的后端服务开发，使用 Python / FastAPI
2. 负责数据库表设计、API 接口开发与文档编写
3. 参与 RAG 检索模块的集成和维护
4. 编写单元测试和集成测试，保障代码质量
5. 参与 CI/CD 流程维护

## 任职要求

1. 计算机、软件工程等相关专业本科及以上
2. 熟练使用 Python，有后端项目开发经验
3. 熟悉 FastAPI 或 Flask，了解 RESTful API 设计规范
4. 了解数据库基础（PostgreSQL / MySQL），能写基本 SQL
5. 了解 Git 和 GitHub 协作开发流程
6. 了解 Docker 基本概念

## 加分项

- 有 LangChain、向量数据库等 AI 相关技术使用经验
- 了解 RAG 基本原理
- 有 GitHub Actions 或 CI/CD 配置经验
- 了解 Linux 服务器基本操作
- 有开源项目贡献经历
