# 岗位 JD：RAG 工程师实习生

> 说明：本文档为脱敏示例 JD。

## 基本信息

- 公司：YY 数据科技（示例，约 100 人，企业知识管理方向）
- 岗位：RAG 工程师实习生
- 地点：北京
- 类型：实习（每周 4 天以上，至少 4 个月）

## 岗位职责

1. 参与公司知识库 RAG 系统的开发和优化
2. 负责文档解析、文本切分、向量检索模块的迭代
3. 参与检索质量评估——设计评估指标、构建测试集、分析 bad case
4. 优化 chunk 策略和 recall 策略，提升检索命中率
5. 参与 Prompt 调优，改善生成内容的准确性和可读性

## 任职要求

1. 计算机、软件工程等相关专业本科及以上
2. 熟练使用 Python，能独立完成模块开发和测试
3. 了解 RAG 系统的完整 pipeline（文档处理 → 向量化 → 检索 → 生成）
4. 使用过至少一种向量数据库（Chroma、FAISS、Milvus 等）
5. 了解 Git 版本控制
6. 能阅读英文技术文档

## 加分项

- 有 LangChain / LlamaIndex 等框架使用经验
- 了解文本切分策略（chunk size、overlap、语义切分）
- 了解 Embedding 模型的选择和评估
- 有 FastAPI 后端开发经验
- 了解 Rerank 和混合检索
